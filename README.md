# Spotify Clone

This is a simple Spotify-like music player created using HTML, CSS, and JavaScript.

## Features
- Play/pause songs
- Previous/Next controls
- Song list display

## Technologies Used
- HTML
- CSS
- JavaScript

## How to Run
- Download or clone this repo
- Open `index.html` in your browser
